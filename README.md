# BitFeeder

## Project Name
BitFeeder

## What the app does
BitFeeder is a responsive web app for YouTube creators that turns rough scripts into performance-friendly chunks for easier recording. It is built for comedy timing, commentary videos, explainer content, and semi-improvised delivery rather than generic business teleprompting.

## Core Idea
Help creators shape messy draft writing into readable beats that are easier to perform on camera without losing rhythm, personality, or natural delivery.

## Core MVP Features
- Paste or write a rough script.
- Break long text into short performance-friendly chunks.
- Let creators adjust chunk size and pacing manually.
- Provide a clean reading view for recording.
- Support quick switching between editing mode and recording mode.
- Work well across desktop and mobile screens.

## Target Workflow
Desktop is the main workspace for writing, editing, and chunking scripts. Phone is the main recording companion for reading the chunked script during filming.

## Tech Stack
- React
- TypeScript
- Vite

## Design Philosophy
Keep the interface minimal, fast, and creator-focused. The spoken text should always be the most important visual element, with controls that stay out of the way and support confident performance.

## Build Strategy In Phases
### Phase 1
Establish the core editor and chunking workflow with a simple responsive UI.

### Phase 2
Add a polished reading mode optimized for phone-based recording and performance pacing.

### Phase 3
Refine creator quality-of-life features, improve responsiveness, and prepare the app for broader usage.
